# User Tracker

A simple User Tracker web application built using HTML, CSS, JavaScript, and Tailwind CSS.

## Features

- Dynamic user card generation using JavaScript
- Real-time search functionality
- Responsive design
- Modern dark-themed UI
- Blur overlay card effect
- Fast and lightweight

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Tailwind CSS

## How It Works

User data is stored in a JavaScript array. Cards are created dynamically using DOM manipulation. As the user types in the search box, the application filters profiles and updates the displayed results instantly.
